## 1. 项目开发准备
    1）描述项目
    2）技术选型
    3）API接口/接口文档/测试接口

## 2. 启动项目开发
    1） 使用React脚手架创建项目
    2） 开发环境运行：npm start
    3） 生产环境打包运行：
        ```
        npm run build      
        server build
        ```

## 3. git管理项目
    1） 创建远程仓库
    2） 创建本地仓库
        a. 配置.gitignore
        b. git init
        c. git add
        d. git commit -m 'init'
    3）将本地仓库推送到远程仓库
        git remote add origin url(替换)
        git push origin main(替换)
